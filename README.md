# Who are u?

Demonstrates NPM package writting best practice!

## install 
```
npm i who-are-u
```

usage
```
import WhoAreU, { whoareu } from 'who-are-you';

const whoAreU = WhoAreU('man');

console.log(whoAreU); // Hey man!

const whoru = whoareu('woman');

console.log(whoru); // Hey woman!

```
